# ```Color-Flipper```
```Flip the color of an element using svelte.js```

A simple [Svelte.js](https://svelte.dev/)  project that allows users to change the color of a circle by clicking on different buttons. This project demonstrates basic reactive behavior in Svelte and showcases how to handle user input to dynamically update the DOM.
